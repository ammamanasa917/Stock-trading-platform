# Stock Trading Platform

A Java-based console application that simulates a basic stock trading system.

## Features
- View available stocks
- Buy and sell stocks
- Manage portfolio
- Check account balance
- Save portfolio using file handling

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File Handling

## How to Run
```bash
javac *.java
java Main
```
